# Sessions

Paused problem sessions live here. When you need to set aside an active problem to work on something else, move `active-problem.md` and `active-solution.cs` into a subfolder here.

## Structure

Each paused session gets its own folder named after the problem:

```
sessions/
  5-longest-palindromic-substring/
    active-problem.md
    active-solution.cs
```

## Resuming a Session

When ready to resume, move the files back to repo root and continue. The AI will detect the existing active files on next problem paste and offer to resume.
